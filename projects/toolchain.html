Toolchain Extension

Synopsis

  • Extended an in-house engine; developing a toolchain to automatically import assets (textures, meshes, audio, materials, etc)
  • Built a tool to recursively scan an asset directory for .descriptor files - files which specified import settings (asset type, compression settings, shader/texture linking, etc) - and use them to process source asset files into binary chunkfiles.
  • Created an integrated runtime asset loader to resolve chunkfiles into usable game assets at runtime.
  • Stored and managed asset references in the editor using a URI-based manifest system.
  • Used C++ for tool creation and integration with the engine.T
  • Collaborated closely with artists to test and gain feedback on the toolchain; creating documentation to assist them in using it.
